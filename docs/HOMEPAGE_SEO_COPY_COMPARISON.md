# Homepage SEO Copy Comparison

This document is for **review only**.

- Scope: **main homepage only**
- Goal: **conversion-first SEO**
- Status: **no page changes implemented**
- Review mode: compare current vs proposed copy side by side, then approve item by item

## Keyword Direction

### Primary keyword cluster
- `skincare coreano`
- `rutina de skincare coreano`
- `caja de skincare coreano`
- `suscripción de skincare coreano`
- `skincare coreano México`

### Conversion-first positioning
Rather than optimizing for broad discovery alone, the proposed copy leans toward:

- a **curated Korean skincare box**
- a **simple routine**
- **delivered every 2 months**
- **for buyers in Mexico**

This should attract visitors closer to purchase intent, not just curiosity traffic.

## 1. Homepage Metadata

### 1.1 Title Tag

**Priority:** Recommended change  
**Type:** SEO + CTR

**Current**

```text

```

**Proposed**

```text
Skincare coreano en México | KumiBox
```

**Why this helps**

- Adds the main keyword early
- Makes the offer easier to understand in search results
- Keeps the brand visible without wasting the whole title on brand only

```text

```

## 5. FAQ Rewrites

### 5.1 Compatibilidad con tu piel

**Priority:** Recommended change  
**Type:** SEO + conversion objection handling

**Current**

```text
Cada caja está diseñada como una rutina universal, equilibrada y suave, pensada para la mayoría de pieles sanas. En lugar de complejidad, priorizamos compatibilidad y claridad.
```

**Proposed**

```text
KumiBox está pensada para pieles sanas que buscan una rutina de skincare coreano más simple y compatible. Priorizamos fórmulas suaves y fáciles de integrar, no tratamientos agresivos ni combinaciones difíciles de seguir.
```

**Why this helps**

- Stronger keyword alignment
- Better defines the right customer
- Reduces ambiguity without overclaiming personalization

### 5.2 Tamaño completo o muestras

**Priority:** Recommended change  
**Type:** Conversion objection handling

**Current**

```text
Los tamaños pueden variar. Lo importante es la duración: cada producto está pensado para durar hasta que llegue tu siguiente caja con un uso normal. Nunca recibirás dos productos de un solo uso.
```

**Proposed**

```text
Los tamaños pueden variar según la edición, pero cada caja está pensada para acompañarte hasta tu siguiente envío. La idea no es llenarte de muestras, sino darte una rutina útil y duradera.
```

**Why this helps**

- More natural, buyer-centered phrasing
- Better addresses a common hesitation

### 5.3 Qué pasa si un producto no te funciona

**Priority:** Recommended change  
**Type:** Trust

**Current**

```text
Priorizamos skincare suave y de uso diario para minimizar irritación e incompatibilidad. Si algo no te convence, escríbenos: estamos para ayudarte.
```

**Proposed**

```text
Seleccionamos productos de uso diario pensados para minimizar incompatibilidades dentro de la rutina. Si algún producto no te convence, puedes escribirnos y te ayudamos a revisar cómo integrarlo o si conviene pausarlo.
```

**Why this helps**

- More credible than generic reassurance
- Feels more supportive and specific

### 5.4 Cuándo se envía tu caja

**Priority:** Recommended change  
**Type:** SEO + conversion

**Current**

```text
Las cajas se envían cada 2 meses en una fecha previsible. Recibirás el seguimiento en cuanto tu caja esté en camino.
```

**Proposed**

```text
Tus cajas se envían cada 2 meses según la ventana de entrega que te corresponda. Antes de comprar puedes revisar tus próximas fechas estimadas directamente en la página.
```

**Why this helps**

- Connects FAQ to the new delivery-windows section
- Improves clarity before purchase

### 5.5 Compromiso y cancelación

**Priority:** Recommended change  
**Type:** Conversion objection handling

**Current**

```text
Sí. Puedes cancelar cuando quieras o saltarte un mes si lo necesitas: sin penalizaciones ni preguntas.
```

**Proposed**

```text
Sí. Puedes cancelar cuando quieras o pausar tu siguiente envío si lo necesitas. Queremos que sientas que tu rutina se adapta a ti, no al revés.
```

**Why this helps**

- Slightly stronger emotional reassurance
- Better fit with premium conversion language
